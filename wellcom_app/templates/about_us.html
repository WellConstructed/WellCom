Why WellCom?

319 million people in sub-Saharan Africa lack access to clean drinking water.

Handpump wells are a cost-effective way to bring clean water to remote areas. However, within the first year, almost half of handpumps break down, and at any given time, one third of rural handpumps in sub-Saharan Africa are nonfunctional.

WellCom enables aid organizations to remotely monitor their wells by providing automated usage statistics via cellular networks. This data enables organizations to respond quickly when a well breaks, optimize operations, and quantify the impact of their work.

How it works

WellCom's hardware is made using materials that are both easily available and cost effective. The monitoring system is self contained and can function for long periods of time without any maintenance.

WellCom is an open source project meaning all of our designs are available for your viewing and replication. Necessary materials includes:

  • Arduino Uno
  • FONA gsm module
  • 3.7v 1200mAh Battery
  • 5v Solar Panel
  • USB/DC/Solar Lithium Ion/Polymer charger kit

For a more detailed parts list and construction schematic, please visit the Github Hardware Repository. The source code for both the Wellcom App and the arduino can be found in the Github Software Repository. The monitoring system schematic is drawn using fritzing, a free software which can be found at fritzing.org.
